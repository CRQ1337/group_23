 1. Создать файл “new.json” 
>>> touch new.json
 
2. Добавить файл под гит. 
>>> git add new.json либо git add .
 
3. Закоммитить файл. 
>>> git commit -m 'added new.json'
 
4. Отправить файл на внешний GitHub репозиторий. 
>>> git push
